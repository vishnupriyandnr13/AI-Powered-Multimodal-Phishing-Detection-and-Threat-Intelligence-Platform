# 🛡️ AI-Powered Multimodal Phishing Detection and Threat Intelligence Platform

An intelligent cybersecurity platform that leverages **Generative AI** and **multimodal analysis** to detect phishing attacks from **URLs, emails, and images** while providing centralized **threat intelligence**, **incident response**, and **security monitoring** capabilities.

Built using **React**, **TypeScript**, **Firebase**, and **Google Gemini AI**, the platform enables organizations and security teams to identify, assess, and manage phishing threats through an intuitive, AI-driven interface.

---

## 🌟 Key Features

- 🤖 AI-powered phishing detection using Large Language Models (LLMs)
- 🌐 URL analysis for malicious and suspicious websites
- 📧 Email content analysis for phishing indicators
- 🖼️ Image and screenshot-based phishing detection using multimodal AI
- 📊 Threat intelligence dashboard with security insights
- 🚨 Incident response and case management system
- 📈 Risk scoring and confidence-based threat assessment
- 🔥 Firebase-powered backend for real-time data management
- 🎨 Modern and responsive user interface built with React and Tailwind CSS
- 🔒 Secure architecture with Firestore security rules and role-based access support

  

## 🛠️ Technology Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Artificial Intelligence:** Google Gemini API
- **Backend & Database:** Firebase Firestore
- **Charts & Visualization:** Recharts
- **Icons:** Lucide React
- **Routing:** React Router

---

## 🎯 Project Objective

The objective of this project is to enhance cybersecurity by combining **Generative AI**, **multimodal input processing**, and **threat intelligence** into a unified platform capable of identifying phishing attempts across multiple communication channels.

By analyzing **URLs**, **email content**, and **images**, the system provides actionable insights, automated risk assessment, and streamlined incident management to help users respond to phishing threats more effectively.

### Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
